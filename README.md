# pywebanim

Embed Manim animations into HTML effortlessly.

## ✨ Features
- Render Manim scenes to video
- Automatically generate HTML embeds
- Simple CLI interface

## 📦 Installation

```bash
pip install pywebanim
```
## 🚀 Usage
```bash
pywebanim scene.py MyScene
```
## 🧪 Example
```python
from pywebanim import render_to_html

render_to_html("scene.py", "MyScene")

```
